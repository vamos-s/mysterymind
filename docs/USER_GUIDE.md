## 2. 게임 카테고리 (Game Categories)

### 🔍 미스터리 (Mystery)
> 범인을 추리하는 스토리형 탐정 게임

**플레이 방법:**
1. 사건의 상황과 단서를 읽습니다
2. 용의자들의 알리바이를 확인합니다
3. 증거를 수집하고 단서를 분석합니다
4. 범인을 지정하여 해결합니다

### 👀 탐색 (Search) - Coming Soon
> 숨겨진 물건을 찾는 관찰력 게임

**플레이 방법:**
1. 제시된 물건 목록을 확인합니다
2. 장면 속에서 숨겨진 물건을 찾습니다
3. 시간 제한 내에 모든 물건을 찾습니다

### 🔒 탈출 (Escape) - Coming Soon
> 퍼즐을 풀고 방에서 탈출하는 게임

**플레이 방법:**
1. 방을 탐색하고 단서를 찾습니다
2. 퍼즐을 해결하여 열쇠를 얻습니다
3. 모든 퍼즐을 풀어 탈출합니다

### 🖼️ 그림 (Picture) - Coming Soon
> 그림에서 다른 점을 찾는 게임

**플레이 방법:**
1. 두 그림을 비교합니다
2. 다른 부분을 모두 찾습니다
3. 시간 내에 모든 차이점을 찾습니다

## 3. 미스터리 게임 상세 가이드 (Mystery Game Guide)

### 📊 게임 구성 (Game Structure)

각 레벨은 다음 요소로 구성됩니다:

| 요소 | 설명 |
|------|------|
| **시나리오** | 사건의 상황 설명 |
| **용의자 (4명)** | 용의자 정보, 진술, 알리바이 |
| **단서 (5개)** | 사건을 해결하는 힌트 |
| **증거 (4개)** | 포인트로 수집 가능한 중요 정보 |
| **타임라인** | 용의자들의 시간별 위치 및 활동 |

### 🎯 플레이 순서 (Play Order)

```
1. 시나리오 읽기
   ↓
2. 용의자 정보 확인
   ↓
3. 타임라인 분석
   ↓
4. 단서 연구
   ↓
5. 증거 수집 (선택사항)
   ↓
6. 노트 작성 (선택사항)
   ↓
7. 범인 지정
   ↓
8. 결과 확인
```

### ⏱️ 타이머 시스템 (Timer System)

| 난이도 | 시간 제한 | 타이머 동작 |
|--------|----------|-------------|
| Easy (Lv1~8) | 10분 | 범인 지정 시 정지 |
| Medium (Lv9~26) | 8분 | 범인 지정 시 정지 |
| Hard (Lv27~50) | 8분 | 범인 지정 시 정지 |

- 시간이 다 되면 자동으로 게임 종료
- 답을 선택하면 즉시 타이머 정지

### 💡 힌트 시스템 (Hint System)

| 난이도 | 힌트 횟수 |
|--------|----------|
| Easy (Lv1~8) | 4회 |
| Medium (Lv9~26) | 3회 |
| Hard (Lv27~50) | 2회 |

힌트 사용법:
1. 하단의 "힌트 보기" 버튼 클릭
2. 힌트 포인트가 차감됨 (50점)
3. 단계별 힌트가 표시됨

### 📈 레벨 구성 (Level Structure)

| 레벨 | 난이도 | 특징 |
|------|--------|------|

| 항목 | 점수 |
|------|------|

| 힌트 사용 | -50점/회 |
| 증거 수집 | -각 증거 비용 |
| 잔여 시간 | +초당 1점 |
| 콤보 보너스 | +50~200점 (연속 정답 시) |

## 5. 게임 시스템 (Game Systems)

### 🕐 타임라인 시스템 (Timeline System)

**개요:** 각 용의자의 시간별 위치와 활동을 시각화

**기능:**
- 📋 시간별 테이블 형태 표시
- 🔴 범죄 발생 시간대 빨간색 하이라이트
- ✓ 검증된 알리바이 체크마크 표시
- 📱 반응형 스크롤 가능 테이블

**사용법:**
```
1. 범죄 시간 확인 (빨간색 표시)
2. 해당 시간대 각 용의자의 위치 확인
3. 알리바이가 모순되는 용의자 추적
4. 시간 흐름을 따라 용의자의 행동 재구성
```

### 🛡️ 알리바이 검증 시스템 (Alibi Verification System)

**개요:** 용의자의 알리바이가 사실인지 확인

**배지 종류:**
| 배지 | 의미 | 색상 |
|------|------|------|
| ✓ Alibi Verified | 알리바이 확인됨 | 초록색 |
| ⚠️ Alibi Contradicted | 알리바이 모순됨 | 노란색 |
| (없음) | 알리바이 미검증 | 기본 |

**사용법:**
```
1. 용의자 카드에 표시된 알리바이 배지 확인
2. 단서와 증거로 알리바이 검증
3. 모순된 알리바이는 범인 가능성 높음
```

### 📦 증거 수집 시스템 (Evidence Collection System)

**개요:** 포인트를 소모하여 중요한 증거 수집

**증거 구조:**
- **ID**: 증거 고유 번호
- **이름 (Name)**: 증거 이름
- **아이콘 (Icon)**: 이모지 아이콘
- **설명 (Description)**: 수집 후 공개되는 상세 정보
- **비용 (Cost)**: 수집에 필요한 포인트
- **위치 (Location)**: 증거 발견 장소

**사용법:**
```
1. 증거 카드에서 비용 확인
2. 보유 포인트가 충분하면 수집 버튼 클릭
3. 포인트가 차감되고 상세 설명 표시
4. 수집된 증거는 초록색 하이라이트
```

**비용 예시:**
| 증거 | 비용 |
|------|------|
| CCTV 영상 | 35점 |
| 은행 기록 | 40점 |
| 출입 로그 | 30점 |
| 목격자 진술 | 35점 |

### 📝 노트/메모 시스템 (Notes System)

**개요:** 각 용의자별로 추리 내용 정리

**기능:**
- 용의자별 개별 메모 영역
- 자유로운 텍스트 입력
- 정답 공개 후 편집 불가
- 애니메이션으로 접기/펼치기

**사용법:**
```
1. "Show Notes" / "Hide Notes" 버튼으로 토글
2. 각 용의자의 메모 영역에 추리 내용 작성
3. 단서, 의심점, 결론 등 정리
4. 정답 공개 후 메모는 읽기 전용
```

**노트 작성 팁:**
```
용의자: David
- 알리바이: 도서관에 있었다고 주장
- 단서: 3:10 PM에 도서관 CCTV 없음
- 증거: 3:12 PM에 가방을 가지고 나옴
- 의심: 알리바이 입증 불가
```

## 6. 팁과 전략 (Tips & Strategies)

### 🎯 정답 추리 팁 (Deduction Tips)

1. **타임라인 우선 분석**
   - 범죄 시간대에 현장에 없는 용의자는 범인이 아님
   - 알리바이가 모순되는 용의자에 집중

2. **단서 철저히 검토**
   - 모든 단서를 읽고 연결 고리 찾기
   - 시간적 순서로 단서 정리

3. **증거 효율적 수집**
   - 중요한 증거부터 수집
   - 포인트 계획적으로 사용

4. **노트 적극 활용**
   - 각 용의자별로 체계적으로 정리
   - 결론 도출 과정 기록

### 📊 포인트 관리 전략 (Point Management)

**포인트 획득:**
- 정답: 500~600점
- 잔여 시간: 초당 1점
- 콤보 보너스: 연속 정답 시

**포인트 사용:**
- 힌트: 50점/회
- 증거 수집: 30~40점/개

**전략:**
```
초기 포인트: 1000점
├─ 힌트 1회 사용: -50점 → 950점
├─ 증거 2개 수집: -75점 → 875점
├─ 정답 맞춤: +550점 → 1425점
└─ 잔여 시간 5분: +300점 → 1725점
```

### 🎓 고급 플레이어 팁 (Advanced Tips)

1. **단서 유형 이해**
   - `time`: 범죄 시간 정보
   - `alibi`: 알리바이 관련
   - `evidence`: 증거 관련

2. **단서 타입 활용**
   - `relatedSuspect` 필드로 연관 용의자 빠르게 파악

3. **콤보 보너스 노리기**
   - 여러 레벨 연속 정답으로 보너스 획득

4. **효율적 힌트 사용**
   - 막혔을 때만 사용
   - 마지막 힌트는 아껴두기

## 7. 자주 묻는 질문 (FAQ)

### ❓ 일반 질문 (General Questions)

**Q: 게임을 시작하는 방법은?**
A: 메인 페이지에서 원하는 게임 카테고리를 클릭하세요.

**Q: 언어를 변경하는 방법은?**
A: 화면 상단의 언어 선택 버튼에서 원하는 언어를 선택하세요.

**Q: 점수는 어떻게 계산되나요?**
A: 정답 점수 + 잔여 시간 점수 + 콤보 보너스 - 힌트/증거 비용

**Q: 레벨은 어떻게 진행되나요?**
A: 정답을 맞히면 다음 레벨로 자동 진행됩니다.

### ❓ 미스터리 게임 질문 (Mystery Game Questions)

**Q: 타임라인은 어떻게 읽나요?**
A: 왼쪽에서 오른쪽으로 시간이 흐릅니다. 범죄 시간은 빨간색으로 표시됩니다.

**Q: 증거를 수집하면 점수가 깎이나요?**
A: 네, 증거 수집 비용만큼 점수가 차감됩니다. 하지만 정답 추리에 큰 도움이 됩니다.

**Q: 노트는 정답 후에도 편집할 수 있나요?**
A: 아니요, 정답이 공개되면 노트는 읽기 전용이 됩니다.

**Q: 힌트는 언제 사용해야 하나요?**
A: 막혔을 때 사용하세요. 힌트 횟수는 난이도별로 제한되어 있습니다.

**Q: 정답을 맞히지 못하면 어떻게 되나요?**
A: "Case Closed" 메시지가 표시되고 게임이 종료됩니다. 다시 도전할 수 있습니다.

### ❓ 기술적 문제 (Technical Issues)

**Q: 게임이 로딩되지 않아요**
A: 인터넷 연결을 확인하고 페이지를 새로고침하세요.

**Q: 점수가 저장되지 않아요**
A: 쿠키가 차단되어 있지 않은지 확인하세요.

**Q: 모바일에서도 플레이할 수 있나요?**
A: 네, 모든 기기에서 플레이할 수 있습니다.
