## 2. 게임 카테고리 (Game Categories)

**플레이 방법:**
1. 사건의 상황과 단서를 읽습니다
2. 용의자들의 알리바이를 확인합니다
3. 증거를 수집하고 단서를 분석합니다
4. 범인을 지정하여 해결합니다

2. 장면 속에서 숨겨진 물건을 찾습니다
3. 시간 제한 내에 모든 물건을 찾습니다

**플레이 방법:**
1. 방을 탐색하고 단서를 찾습니다
2. 퍼즐을 해결하여 열쇠를 얻습니다
3. 모든 퍼즐을 풀어 탈출합니다

### 🖼️ 그림 (Picture) - Coming Soon
> 그림에서 다른 점을 찾는 게임

**플레이 방법:**
1. 두 그림을 비교합니다

## 3. 미스터리 게임 상세 가이드 (Mystery Game Guide)

### 📊 게임 구성 (Game Structure)

각 레벨은 다음 요소로 구성됩니다:

| 요소 | 설명 |
|------|------|
| **시나리오** | 사건의 상황 설명 |
| **용의자 (4명)** | 용의자 정보, 진술, 알리바이 |
| **단서 (5개)** | 사건을 해결하는 힌트 |
| **증거 (4개)** | 포인트로 수집 가능한 중요 정보 |
| **타임라인** | 용의자들의 시간별 위치 및 활동 |

### 🎯 플레이 순서 (Play Order)

```
1. 시나리오 읽기
   ↓
2. 용의자 정보 확인
   ↓
3. 타임라인 분석
   ↓
4. 단서 연구
   ↓
5. 증거 수집 (선택사항)
   ↓
6. 노트 작성 (선택사항)
   ↓
7. 범인 지정
   ↓
8. 결과 확인
```

### ⏱️ 타이머 시스템 (Timer System)

| 난이도 | 시간 제한 | 타이머 동작 |
|--------|----------|-------------|
| Easy (Lv1~8) | 10분 | 범인 지정 시 정지 |
| Medium (Lv9~26) | 8분 | 범인 지정 시 정지 |
| Hard (Lv27~50) | 8분 | 범인 지정 시 정지 |

- 시간이 다 되면 자동으로 게임 종료
- 답을 선택하면 즉시 타이머 정지

### 💡 힌트 시스템 (Hint System)

| 난이도 | 힌트 횟수 | 비용 |
|--------|----------|------|
| Easy (Lv1~8) | 4회 | -20점 |
| Medium (Lv9~26) | 3회 | -20점 |
| Hard (Lv27~50) | 2회 | -20점 |

**힌트 사용법:**
1. 용의자 목록 하단 중앙의 "💡 Get Hint (-20 pts)" 버튼 클릭
2. 힌트 포인트가 차감됨 (-20점)
3. 단계별 힌트가 표시됨

**힌트 카운터:**
- 상단 GameHeader에 "💡 0/2", "💡 1/2" 형태로 표시
- 사용 가능한 힌트 횟수를 한눈에 확인

## 4. 점수 시스템 (Scoring System)

### 🎯 점수 계산 (Score Calculation)

| 항목 | 점수 | 비고 |
|------|------|------|
| 정답 | **+50점** | 범인을 올바르게 지정 |
| 오답 | **-50점** | 잘못된 범인 지정 |
| 힌트 사용 | **-20점** | 단서 힌트 표시 |
| 증거 수집 | **-각 증거 비용** | 증거마다 다름 (30~40점) |

### ⚠️ 오답 패널티 시스템 (Wrong Answer Penalty)

**Level 1 (초급):**
- 시작 점수: 0점
- 오답 시: **-50점** (음수 점수 허용)
- 리셋 없음: 오답을 선택해도 다시 선택 가능
- 패널티 메시지: 표시 안 함 (1단계에서는 -50 할 수 없음)

**Level 2 이상 (중급/고급):**
- 오답 시: **-50점** 차감
- **리셋 조건 1**: 현재 점수 < 50점인 상태에서 오답 → **즉시 Level 1로 리셋**
- **리셋 경고**: 현재 점수가 50점 미만일 때 경고 메시지 표시
  - 예: 점수 49점 → 오답 시 -50점 → -1점 (리셋됨)
  - 예: 점수 50점 → 오답 시 -50점 → 0점 (리셋 안 됨)

**리셋 경고 메시지:**
```
⚠️ Warning: Your score is less than 50 points.
If you answer incorrectly, you will be reset to Level 1!
```

### 📈 점수 관리 전략 (Point Management Strategy)

**추천 점수 관리:**
```
초기 점수: 50점 이상 유지 권장
├─ 힌트 사용: -20점 → 30점
├─ (리셋 위험! 50점 미만)
├─ 오답 선택: -50점 → -20점
└─ ❌ 리셋됨 → Level 1로 돌아감
```

**안전한 플레이:**
```
초기 점수: 100점 이상 권장
├─ 힌트 1회 사용: -20점 → 80점
├─ 증거 1개 수집: -35점 → 45점
├─ (여전히 50점 미만! 주의)
├─ 힌트 2회 사용: -20점 → 25점
└─ (매우 위험! 한 번의 오답으로 리셋)
```

## 5. 레벨 시스템 (Level System)

### 📊 레벨 구성 (Level Structure)

| 레벨 | 난이도 | 특징 |
|------|--------|------|
| Lv1 | Easy | 단순한 시나리오, 명확한 단서, 음수 점수 허용 |
| Lv2~Lv8 | Easy | 복잡한 시나리오, 50점 미만 시 리셋 위험 |
| Lv9~Lv26 | Medium | 복잡한 시나리오, 추가 단서 |
| Lv27~Lv50 | Hard | 어려운 사건, 여러 용의자와 증거 |

### 🔄 레벨 진행 및 리셋 (Level Progress & Reset)

**정답 시:**
- 다음 레벨로 자동 진행
- 정답을 맞히면 화면 하단으로 스크롤
- 게임 완료 시 "Next Level" 버튼 표시

**오답 시 리셋 (Level 2+):**
- 조건: 현재 점수 < 50점
- 동작: 즉시 Level 1로 리셋
- 초기화 내용:
  - 레벨 → 1
  - 현재 문제 → Level 1 문제
  - 점수 → 초기값 유지
  - 사용한 힌트 → 0회
  - 수집된 증거 → 초기화
  - 노트 내용 → 초기화
  - 화면 → 상단으로 스크롤

**Try Again 버튼 (재시작):**
- 게임 완료 후 표시
- 클릭 시 현재 레벨 재시작
- 화면 상단으로 스크롤

## 6. 게임 시스템 (Game Systems)

### 🕐 타임라인 시스템 (Timeline System)

**개요:** 각 용의자의 시간별 위치와 활동을 시각화

**기능:**
- 📋 시간별 테이블 형태 표시
- 🔴 범죄 발생 시간대 빨간색 하이라이트
- ✓ 검증된 알리바이 체크마크 표시
- 📱 반응형 스크롤 가능 테이블

**사용법:**
```
1. 범죄 시간 확인 (빨간색 표시)
2. 해당 시간대 각 용의자의 위치 확인
3. 알리바이가 모순되는 용의자 추적
4. 시간 흐름을 따라 용의자의 행동 재구성
```

### 🛡️ 알리바이 검증 시스템 (Alibi Verification System)

**개요:** 용의자의 알리바이가 사실인지 확인

**배지 종류:**
| 배지 | 의미 | 색상 |
|------|------|------|
| ✓ Alibi Verified | 알리바이 확인됨 | 초록색 |
| ⚠️ Alibi Contradicted | 알리바이 모순됨 | 노란색 |
| (없음) | 알리바이 미검증 | 기본 |

**사용법:**
```
1. 용의자 카드에 표시된 알리바이 배지 확인
2. 단서와 증거로 알리바이 검증
3. 모순된 알리바이는 범인 가능성 높음
```

### 📦 증거 수집 시스템 (Evidence Collection System)

**개요:** 포인트를 소모하여 중요한 증거 수집

**증거 구조:**
- **ID**: 증거 고유 번호
- **이름 (Name)**: 증거 이름
- **아이콘 (Icon)**: 이모지 아이콘
- **설명 (Description)**: 수집 후 공개되는 상세 정보
- **비용 (Cost)**: 수집에 필요한 포인트
- **위치 (Location)**: 증거 발견 장소

**사용법:**
```
1. 증거 카드에서 비용 확인
2. 보유 포인트가 충분하면 수집 버튼 클릭
3. 포인트가 차감되고 상세 설명 표시
4. 수집된 증거는 초록색 하이라이트
```

**비용 예시:**
| 증거 | 비용 |
|------|------|
| CCTV 영상 | 35점 |
| 은행 기록 | 40점 |
| 출입 로그 | 30점 |
| 목격자 진술 | 35점 |

**Disabled 상태:**
- 포인트가 부족하면 수집 버튼 비활성화
- 마우스 호버 시 색상 변화 없음
- 회색 테두리로 명확히 구분

### 📝 노트/메모 시스템 (Notes System)

**개요:** 각 용의자별로 추리 내용 정리

**기능:**
- 용의자별 개별 메모 영역
- 자유로운 텍스트 입력
- 정답 공개 후 편집 불가
- 모달 형태로 표시
- 모달 열릴 때 페이지 스크롤 막기

**사용법:**
```
1. 용의자 목록 하단 중앙의 "📝 Show Notes" 버튼 클릭
2. 모달로 노트 패널 표시 (페이지 스크롤 막힘)
3. 각 용의자의 메모 영역에 추리 내용 작성
4. 단서, 의심점, 결론 등 정리
5. "Hide Notes" 또는 X 버튼으로 모달 닫기
```

**노트 작성 팁:**
```
용의자: David
- 알리바이: 도서관에 있었다고 주장
- 단서: 3:10 PM에 도서관 CCTV 없음
- 증거: 3:12 PM에 가방을 가지고 나옴
- 의심: 알리바이 입증 불가
```

**모달 특징:**
- 화면 중앙에 팝업 표시
- 배경 어두워짐 (backdrop blur)
- 외부 영역 클릭으로 닫기 가능
- 반응형 디자인 (모바일/데스크톱)

## 7. 팁과 전략 (Tips & Strategies)

### 🎯 정답 추리 팁 (Deduction Tips)

1. **타임라인 우선 분석**
   - 범죄 시간대에 현장에 없는 용의자는 범인이 아님
   - 알리바이가 모순되는 용의자에 집중

2. **단서 철저히 검토**
   - 모든 단서를 읽고 연결 고리 찾기
   - 시간적 순서로 단서 정리

3. **증거 효율적 수집**
   - 중요한 증거부터 수집
   - 포인트 계획적으로 사용 (50점 미만 조심!)

4. **노트 적극 활용**
   - 각 용의자별로 체계적으로 정리
   - 결론 도출 과정 기록

### 📊 포인트 관리 전략 (Point Management)

**안전한 플레이:**
```
초기 점수: 100점 이상 유지
├─ 힌트 1회: -20점 → 80점
├─ 증거 1개: -35점 → 45점 (위험! 50점 미만)
└─ 정답: +50점 → 95점 (안전)
```

**위험한 플레이 (피해야 할 패턴):**
```
초기 점수: 50점
├─ 힌트 1회: -20점 → 30점
├─ 오답 선택: -50점 → -20점
└─ ❌ 리셋됨 → Level 1로 돌아감
```

**추천 전략:**
1. 정답 후 점수를 100점 이상으로 유지
2. 힌트는 정말 필요할 때만 사용
3. 50점 미만일 때는 오답 절대 선택 금지
4. 노트에 추리 과정 철저히 기록

### 🎓 고급 플레이어 팁 (Advanced Tips)

1. **단서 유형 이해**
   - `time`: 범죄 시간 정보
   - `alibi`: 알리바이 관련
   - `evidence`: 증거 관련

2. **단서 타입 활용**
   - `relatedSuspect` 필드로 연관 용의자 빠르게 파악

3. **히트 횟수 관리**
   - 난이도별 힌트 횟수 확인
   - 마지막 힌트는 아껴두기

4. **자동 스크롤 활용**
   - 정답 후 화면이 하단으로 자동 스크롤
   - 결과 확인 후 상단으로 스크롤하여 다음 레벨 준비

## 8. 자주 묻는 질문 (FAQ)

### ❓ 일반 질문 (General Questions)

**Q: 게임을 시작하는 방법은?**
A: 메인 페이지에서 원하는 게임 카테고리를 클릭하세요.

**Q: 언어를 변경하는 방법은?**
A: 화면 상단의 언어 선택 버튼에서 원하는 언어를 선택하세요.

**Q: 점수는 어떻게 계산되나요?**
A: 정답 +50점, 오답 -50점, 힌트 -20점, 증거 비용 차감

**Q: 레벨은 어떻게 진행되나요?**
A: 정답을 맞히면 다음 레벨로 자동 진행됩니다.

### ❓ 점수 시스템 질문 (Scoring Questions)

**Q: Level 1에서 오답을 선택하면 어떻게 되나요?**
A: -50점 차감되지만 리셋되지 않습니다. 1단계에서는 음수 점수가 허용됩니다.

**Q: 점수가 50점 미만이면 무슨 일이 일어나나요?**
A: 이 상태에서 오답을 선택하면 즉시 Level 1로 리셋됩니다.

**Q: 정답을 맞히면 점수가 어떻게 되나요?**
A: +50점이 추가됩니다. 높은 점수를 유지하는 것이 중요합니다.

**Q: 힌트를 사용하면 점수가 차감되나요?**
A: 네, 힌트마다 -20점이 차감됩니다. 점수가 50점 미만일 때는 힌트 사용을 주의하세요.

### ❓ 미스터리 게임 질문 (Mystery Game Questions)

**Q: 타임라인은 어떻게 읽나요?**
A: 왼쪽에서 오른쪽으로 시간이 흐릅니다. 범죄 시간은 빨간색으로 표시됩니다.

**Q: 증거를 수집하면 점수가 깎이나요?**
A: 네, 증거 수집 비용만큼 점수가 차감됩니다. 하지만 정답 추리에 큰 도움이 됩니다.

**Q: 노트는 정답 후에도 편집할 수 있나요?**
A: 아니요, 정답이 공개되면 노트는 읽기 전용이 됩니다.

**Q: 힌트 버튼은 어디에 있나요?**
A: 용의자 목록 하단 중앙에 "💡 Get Hint (-20 pts)" 버튼이 있습니다.

**Q: Show Notes는 어떻게 사용하나요?**
A: 용의자 목록 하단 중앙의 "📝 Show Notes" 버튼을 클릭하면 모달로 노트가 표시됩니다.

**Q: 모달이 열려 있을 때 게임 화면을 스크롤할 수 있나요?**
A: 아니요, 모달이 열려 있을 때는 페이지 스크롤이 막힙니다.

**Q: 정답을 맞히지 못하면 어떻게 되나요?**
A: "Case Closed" 메시지가 표시되고 게임이 종료됩니다. 다시 도전할 수 있습니다.

### ❓ 기술적 문제 (Technical Issues)

**Q: 게임이 로딩되지 않아요**
A: 인터넷 연결을 확인하고 페이지를 새로고침하세요.

**Q: 점수가 저장되지 않아요**
A: 쿠키가 차단되어 있지 않은지 확인하세요.

**Q: 모바일에서도 플레이할 수 있나요?**
A: 네, 모든 기기에서 플레이할 수 있습니다.

**Q: 힌트 버튼이 비활성화되어 있어요**
A: 힌트 횟수를 모두 사용했거나, 게임이 종료되었을 때 비활성화됩니다.
